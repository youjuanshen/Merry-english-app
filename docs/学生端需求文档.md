# 学生端需求文档 (PRD)

> 版本：v1.0 | 更新：2026-03-15 | 作者：Merry

---

## 1. 产品概述

### 1.1 产品名称
Merry英语学生端

### 1.2 目标用户
- 福建农村小学三年级学生（27人）
- 年龄：8-9岁
- 设备：iPhone 7、iPhone 7 Plus、iPad

### 1.3 产品目标
通过游戏化的双人合作学习，提高学生英语学习的自我效能感

---

## 2. 设备适配要求

| 设备 | 屏幕尺寸 | 数量 | 优先级 |
|------|----------|------|--------|
| iPhone 7 | 375 x 667 | 2台 | 高 |
| iPhone 7 Plus | 414 x 736 | 4台 | **最高** |
| iPad | 768 x 1024 | 1台 | 中 |

**要求：**
- 所有界面必须在三种设备上正常显示
- 不能有元素重叠、被截断、超出屏幕
- 按钮必须足够大，方便触摸（最小44x44像素）

---

## 3. 页面流程

```
欢迎页(3秒倒计时) → 选择学生 → 开始学习 → 答题界面 → 结果页
```

---

## 4. 页面详细需求

### 4.1 欢迎页面

**功能：**
- 显示App名称和Logo
- 3-2-1倒计时动画
- 倒计时结束后自动跳转到选择学生页面

**验收标准：**
- [ ] 倒计时数字清晰可见
- [ ] 倒计时结束后自动跳转（不卡住）
- [ ] 动画流畅

---

### 4.2 选择学生页面

**功能：**
- 显示全班27名学生名单
- 学生可以选择自己和搭档（共选2人）
- 分页显示，左右滑动切换页面

**布局要求：**

| 设备 | 每页学生数 | 布局 |
|------|-----------|------|
| iPhone 7 (375px) | 9人 | 3列 x 3行 |
| iPhone 7 Plus (414px) | 12人 | 3列 x 4行 |
| iPad (768px) | 20人 | 5列 x 4行 |

**验收标准：**
- [ ] 所有27名学生都能看到（滑动翻页）
- [ ] 学生卡片不重叠、不被截断
- [ ] 选中状态明显（颜色变化）
- [ ] 最多只能选2人
- [ ] 选满2人后"开始学习"按钮可点击
- [ ] 页码指示器正确显示当前页

---

### 4.3 答题界面

**顶部信息栏：**
- 显示当前课程和阶段（如：Unit 1 Lesson 1 - 听力 · 前测）
- 显示进度（如：第3题/共10题）
- 显示两名学生的头像和得分

**题目区域：**
- 根据题型显示不同内容
- 图片居中显示，不被其他元素遮挡

**底部操作区：**
- 选项按钮/输入框
- 提交/下一题按钮

**验收标准：**
- [ ] 题目文字清晰可读
- [ ] 图片完整显示，不被遮挡
- [ ] 选项按钮不重叠
- [ ] 按钮点击区域足够大
- [ ] 音频播放按钮（喇叭）不遮挡图片

---

### 4.4 反馈面板

**触发时机：** 学生选择答案后

**正确时显示：**
- 绿色背景
- "真厉害！" + 正确答案
- "继续"按钮（绿色）

**错误时显示：**
- 红色背景
- "再想想！" + 正确答案
- "继续"按钮（红色）

**验收标准：**
- [ ] 反馈面板从底部滑出（不是跳转新页面）
- [ ] 反馈面板不遮挡题目图片太多
- [ ] "继续"按钮明显可点击
- [ ] 点击"继续"后进入下一题

---

## 5. 题型需求

### 5.1 听力选择题 (listen_select)

**显示内容：**
- 题目文字 + 中文提示
- 喇叭按钮（点击播放音频）
- 图片选项（2-4个）

**交互：**
- 点击喇叭播放音频
- 点击图片选择答案
- 选中后显示反馈

**验收标准：**
- [ ] 喇叭按钮位置合理，不遮挡图片
- [ ] 图片选项大小一致
- [ ] 选中效果明显

---

### 5.2 判断题 (true_false)

**显示内容：**
- 题目文字（如：This is a panda.）
- 一张图片
- YES / NO 两个按钮

**验收标准：**
- [ ] 图片居中显示
- [ ] YES/NO按钮大小一致
- [ ] 按钮不与图片重叠
- [ ] 按钮文字清晰

---

### 5.3 翻转卡片题 (flip_card)

**显示内容：**
- 卡片正面：英文单词
- 卡片背面：中文翻译 + 图片

**交互：**
- 点击卡片翻转
- 翻转动画流畅

**验收标准：**
- [ ] 卡片大小适中
- [ ] 翻转动画流畅
- [ ] 正反面内容清晰

---

### 5.4 气球射击题 (balloon_pop)

**显示内容：**
- 题目（如：Find the tiger）
- 多个上浮的气球，每个气球上有图片

**交互：**
- 气球从底部上浮
- 点击正确的气球得分
- 点击错误的气球扣分

**验收标准：**
- [ ] 气球上浮动画流畅
- [ ] 气球上的图片清晰可见
- [ ] 点击气球有反馈（爆炸效果）

---

### 5.5 情境题 (scenario)

**显示内容：**
- 情境描述（英文 + 中文）
- 情境图片或emoji
- 问题
- 选项

**验收标准：**
- [ ] 情境描述清晰
- [ ] 问题和选项布局合理

---

## 6. 阶段切换

### 6.1 前测阶段 (pretest)

**目的：** 诊断学生水平

**特点：**
- 固定题目顺序
- 不显示难度标签
- 完成后计算学生水平（A/B/C）

### 6.2 实战阶段 (practice)

**目的：** 游戏化练习

**特点：**
- 根据学生水平调整难度
- 显示难度标签（简单/中等/困难）
- 更多游戏化题型（气球、情境等）

**验收标准：**
- [ ] 教师端选择"前测"→ 学生端显示前测题目
- [ ] 教师端选择"实战"→ 学生端显示实战题目
- [ ] 阶段指示器正确显示当前阶段
