# 教师端需求文档 (PRD)

> 版本：v1.0 | 更新：2026-03-15 | 作者：Merry

---

## 1. 产品概述

### 1.1 产品名称
Merry英语教师端（课堂控制台）

### 1.2 目标用户
- 小学英语教师（Merry本人）
- 设备：iPhone 或 iPad

### 1.3 产品目标
帮助教师控制课堂学习活动，实时监控学生进度，记录课堂观察数据

---

## 2. 页面结构

```
首页
├── 课前准备
├── 课中控制
├── 课后反馈
└── 数据看板
    ├── 学生档案
    └── 班级趋势
```

### 3.4 课后反馈 (feedback.html)

**功能模块：**

#### 3.4.1 观察记录表

**表格结构：**

| 姓名 | 掌握经验 | 替代经验 | 社交劝导 | 情绪状态 |
|------|----------|----------|----------|----------|
| 学生1 | 0/1/2/N/A/假 | 0/1/2/N/A/假 | 0/1/2/N/A/假 | 0/1/2/N/A/假 |

**四个维度说明（自我效能感理论）：**

1. **掌握经验**：学生在教师指导下能够独立开始并持续完成任务
2. **替代经验**：学生能在教师示范后模仿语言或动作
3. **社交劝导**：学生在教师鼓励后表现出积极回应
4. **情绪状态**：学生身体姿势和眼神表现出情绪稳定、专注或投入

**评分标准：**
- 0 = 没做到
- 1 = 偶尔做到（1-2次）
- 2 = 经常做到（3次以上）
- N/A = 不适用
- 假 = 请假

**验收标准：**
- [ ] 显示今日观察的7名学生
- [ ] 四个维度列都显示
- [ ] 下拉菜单能选择 0/1/2/N/A/假
- [ ] 有"保存记录"按钮
- [ ] 有补充记录文本框

---

**验收标准：**
- [ ] 各项内容正确显示
- [ ] 趋势图可视化清晰
- [ ] 有"重新生成报告"按钮

## 4. 教师端 ↔ 学生端通信

### 4.1 通信机制
使用 `localStorage` 存储指令，学生端定时轮询读取

### 4.2 指令格式

```javascript
{
    action: 'start' | 'pause' | 'resume' | 'end',
    module: 'listening' | 'reading' | 'writing' | 'speaking',
    phase: 'pretest' | 'practice',
    lesson: 'U1L1',
    timeLimit: 0 | 10 | 15 | 20 | 30,
    timestamp: Date.now()
}
```

### 4.3 验收标准
- [ ] 教师端发出指令后，学生端能在3秒内响应
- [ ] 指令内容正确传递（模块、阶段、时间限制）
