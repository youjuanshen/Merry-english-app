## 二、各模块题型规范

### 听力模块 (listening)

#### 前测 (10题，由简到难)
| 题号 | 题型 | 说明 |
|------|------|------|
| 1-3 | listen_select（听单词选图） | audio=单词，options=4张图片 |
| 4-6 | listen_select（听句子选中文） | audio=英文句子，options=4个中文翻译 |
| 7-8 | listen_tf（听句子判断对错） | audio=英文句子，image=1张图片，correct=true/false |
| 9-10 | listen_select（听长句选中文） | audio=较长英文句子，options=4个中文翻译 |

#### 实战 (20题，覆盖全部课文句子)
混合使用以上3种题型，确保每句课文至少出现1次。

#### 题型格式

```json
// listen_select（听音选图）
{
  "type": "listen_select",
  "audio": "panda",
  "options": [
    "<img src=\"assets/images/panda.png\" width=\"80\">",
    "<img src=\"assets/images/horse.png\" width=\"80\">",
    "<img src=\"assets/images/bear.png\" width=\"80\">",
    "<img src=\"assets/images/duck.png\" width=\"80\">"
  ],
  "correct": 0,
  "chinese": "熊猫"
}

// listen_select（听句选中文）
{
  "type": "listen_select",
  "audio": "This is a bear. It's big.",
  "options": [
    "这是一只熊。它很大。",
    "那是一匹马。",
    "它们很漂亮。",
    "好多动物！"
  ],
  "correct": 0,
  "chinese": "这是一只熊。它很大。"
}

// listen_tf（听音判断对错）
{
  "type": "listen_tf",
  "audio": "This is a bear.",
  "image": "<img src=\"assets/images/bear.png\" width=\"80\">",
  "correct": true,
  "chinese": "这是一只熊，正确！"
}
```

### 阅读模块 (reading)

#### 前测 (10题，由简到难)
| 题号 | 题型 | 说明 |
|------|------|------|
| 1-3 | word_match（看单词选图） | word=英文单词，options=4张图片 |
| 4-6 | sentence_match（看句子选图/选中文） | sentence=英文句子，options=4张图片或4个中文 |
| 7-8 | sentence_match（看句子选中文） | sentence=英文句子，options=4个中文翻译 |
| 9-10 | sentence_match（情景选择） | sentence=中文情景描述，options=4个英文回答 |

**禁止**：前测不出对话补全题（A: ... B: ______）

#### 题型格式

```json
// word_match（看单词选图）
{
  "type": "word_match",
  "word": "bear",
  "sentence": "This is a bear.",
  "chinese": "这是一只熊。",
  "options": [
    {"text": "<img src=\"assets/images/bear.png\" width=\"80\">", "value": "bear"},
    {"text": "<img src=\"assets/images/cat.png\" width=\"80\">", "value": "cat"},
    {"text": "<img src=\"assets/images/dog.png\" width=\"80\">", "value": "dog"},
    {"text": "<img src=\"assets/images/bird.png\" width=\"80\">", "value": "bird"}
  ],
  "correct": "bear"
}

// sentence_match（看句子选中文）
{
  "type": "sentence_match",
  "sentence": "They are beautiful.",
  "options": [
    {"text": "它们很漂亮。", "value": "correct"},
    {"text": "它们很大。", "value": "wrong1"},
    {"text": "它们很可爱。", "value": "wrong2"},
    {"text": "它是黑白色的。", "value": "wrong3"}
  ],
  "correct": "correct"
}
```

### 写作模块 (writing)

#### 前测 (10题)
| 题号 | 题型 | 说明 |
|------|------|------|
| 1-3 | letter_select（首字母选择） | 看中文+图片，选首字母 |
| 4-6 | pinyin_fill / fill_blank（填空） | 补全单词或句子 |
| 7-10 | word_puzzle（拼单词） | 打乱字母拼出单词 |

#### 实战 (25题)
| 题型 | 数量 | 说明 |
|------|------|------|
| sentence_order（词块排列） | 15 | **每句课文都要有**，必须包含 chineseWords 和 chineseScrambled |
| fill_blank（句子填空） | 5 | 英文句子挖空，选词填入 |
| word_puzzle（拼单词） | 5 | 核心词汇拼写 |

#### 题型格式

```json
// sentence_order（双语排列 —— 重点题型！）
{
  "type": "sentence_order",
  "chinese": "这是一只熊。",
  "words": ["This", "is", "a", "bear"],
  "scrambled": ["bear", "a", "This", "is"],
  "chineseWords": ["这", "是", "一只", "熊"],
  "chineseScrambled": ["熊", "这", "一只", "是"]
}

// fill_blank（句子填空）
{
  "type": "fill_blank",
  "prompt": "This is a _____. It's big.",
  "image": "<img src=\"assets/images/bear.png\" width=\"80\">",
  "options": ["bear", "bird", "horse", "panda"],
  "correct": 0,
  "chinese": "这是一只熊。它很大。"
}

// letter_select（首字母选择）
{
  "type": "letter_select",
  "prompt": "熊猫的首字母是？",
  "image": "<img src=\"assets/images/panda.png\" width=\"80\">",
  "options": ["d", "b", "p", "q"],
  "correct": 2,
  "chinese": "熊猫"
}

// word_puzzle（拼单词）
{
  "type": "word_puzzle",
  "chinese": "熊",
  "image": "<img src=\"assets/images/bear.png\" width=\"80\">",
  "letters": ["b", "e", "a", "r"],
  "scrambled": ["a", "r", "b", "e"]
}
```

## 三、铁律（违反任何一条都要重做）

| # | 规则 | 为什么 |
|---|------|--------|
| 1 | audio 字段只能是**纯英文** | 🔊会用英文语音朗读，中文会乱读 |
| 2 | audio 字段不能有 `___` | 🔊会读出下划线 |
| 3 | 图片必须和 audio 内容匹配 | audio说bear，图片必须是bear |
| 4 | 复数句子不用单数图片 | "These are pandas" 不能配一只熊猫的图 |
| 5 | 选项必须4个 | 统一规范 |
| 6 | 正确答案位置不能都在第一个 | 代码会随机打乱，但数据本身也要分散 |
| 7 | 前测不出对话补全 | A: ... B: ______ 太难 |
| 8 | chinese 字段是简洁中文翻译 | 不写指令性文字 |
| 9 | 身体部位词不能单独选图 | "ear"不能配动物图片，要用句子"They have small ears."配图 |
| 10 | 每句课文在每个模块至少出现1次 | 听力+阅读+写作+口语都要覆盖 |

## 四、各课课文内容（必须100%覆盖）

### U1L2: These are pandas.
**词汇**: small, ear, eye, baby, long, leg, these, those
**课文句子**:
1. These are pandas. They have small ears and black eyes.
2. Look, baby pandas. They are lovely.
3. Now let's go and see those horses. All right.
4. Those are horses. They are brown. They have long legs. They like grass.
5. Let's ride a horse. Great! Cool!

### U1L3: I have an animal friend.
**词汇**: jump, mouth, meat, tiger, no, rabbit, carrot
**课文句子**:
1. I have an animal friend. Please guess!
2. Is it a rabbit? Yes, it is.
3. Is it a cat? No, it isn't. It's a tiger!
4. I am white. I can jump. My mouth is small. I have two long ears. I like carrots. What am I?
5. I am black and yellow. I can run fast. My mouth is big. I like meat. What am I?

### U1L4: Look! It's a cat.
**词汇**: draw, nose, face, eye, ear, mouth
**课文句子**:
1. Let's draw together!
2. This is a face. These are two eyes.
3. That is a face. Those are two eyes.
4. Look! It's a cat.
5. These are two ears. This is a nose and a mouth.
6. Look, a nose and a mouth.
7. Is it a panda? No, it isn't.

### U2L1: What size do you wear?
**词汇**: T-shirt, do, wear, size, big, small
**课文句子**:
1. Put on your team T-shirts, please.
2. What size do you wear? Size S / Size M / Size L / Size XL.
3. It's too big. / It's too small.

### U2L2: Whose shirt is this?
**词汇**: whose, shirt, jacket
**课文句子**:
1. Yang Ming, this is your T-shirt.
2. This is my jacket. This is my shirt.
3. Whose shirt is this? It's Yang Ming's shirt.
4. Whose jacket is this?
5. Is this your jacket? No, it isn't. It's Wang Tao's jacket.
6. Here you are. Thank you. You're welcome.

### U2L3: Where is my sweater?
**词汇**: where, room, sweater, under, bed, chair, schoolbag
**课文句子**:
1. Where is my sweater, Mom?
2. Is it on your bed? No, it isn't.
3. Is it under your chair? No, it isn't.
4. Look! Your sweater is in your schoolbag!
5. Dear, put away your clothes, please.
6. Okay, Mom.
7. Oh, thank you very much. You're welcome.

### U2L4: Put on your sports shoes.
**词汇**: sport, shoe, skirt, want, shorts
**课文句子**:
1. I'm sorry, you can't wear your skirt.
2. OK, thank you.
3. Put on your T-shirt and shorts, please.
4. Now put on your sports shoes.
5. They are too big for you. / They are too small for me.
6. I want my sports shoes, too.
7. My sports shoes, please. Ah?!
