## 设计原则（已写入项目规则）

### 命题规范
- 题目说明用中文，考察内容用英文
- 答案不能被提前泄露（中文提示/图片/选项都不能暴露）
- 选择题必须4个选项
- chinese 字段必须是完整中文句子

### 支架撤退（Scaffolding Fading）
- Level 1 全支架：图片 + 英文 + 中文（正确率<60%）
- Level 2 半支架：图片 + 英文（正确率60-80%）
- Level 3 无支架：仅图片或仅英文（正确率>80%）

### 工作流程
- 本地改代码 → localhost 测试 → 每任务 commit → 全部完成统一 push
- Merry 测试用线上版本（GitHub Pages），改完必须 push
