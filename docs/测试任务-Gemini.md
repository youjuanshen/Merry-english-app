## 第三步：执行测试

### 3.1 学生端测试清单

#### 欢迎页面
```
□ 打开网页
□ 检查倒计时 3-2-1 是否正常显示
□ 检查倒计时结束后是否自动跳转
□ 截图记录
```

#### 学生选择页面
```
□ 检查每页显示的学生数量是否符合需求
   - iPhone 7: 9人（3x3）
   - iPhone 7 Plus: 12人（3x4）
   - iPad: 20人（5x4）
□ 检查学生卡片是否有重叠或被截断
□ 滑动翻页是否正常
□ 是否能看到全部27名学生
□ 选择学生的交互是否正常
□ 选满2人后"开始学习"按钮是否可点击
□ 截图记录
```

#### 答题界面
```
□ 题目文字是否清晰
□ 图片是否完整显示
□ 图片是否被喇叭按钮遮挡
□ 选项按钮是否有重叠
□ YES/NO按钮布局是否正确
□ 翻转卡片是否正常工作
□ 截图记录每种题型
```

#### 反馈面板
```
□ 答对后反馈面板是否从底部滑出
□ 反馈面板是否正确显示（不是跳转新页面）
□ "继续"按钮是否可点击
□ 点击"继续"后是否进入下一题
□ 反馈面板是否遮挡太多内容
□ 截图记录
```

#### 阶段切换
```
□ 先在教师端选择"实战"阶段
□ 再在学生端开始学习
□ 检查是否加载实战题目（而不是前测）
□ 截图记录
```

#### 首页

#### 课后反馈

```
□ 截图记录
```
